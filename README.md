# kvSql
简单的KV数据库个人项目（以跳表为索引，跳表最大高度为15）
使用语言为C#， .NET core框架（可跨平台），.NET版本为8.0

### 项目模块
- KV数据库——部分完成
- Raft——开发中
- Rpc——开发中

### 目前功能
- KV数据库支持泛型，目前为了便于运行固定了<string, string>与<string, long>两种使用类型。但是代码可以扩展至其他类型。long类型数据支持基本数学运算（加和减）。
- 数据库以json格式保存

### 跳表读写效率
|数据量(万条)|写耗时(ms)|读耗时(ms)|
|---|---|---|
|1|20|22|
|5|119|125|
|10|191|236|
|50|914|996|
|100|1792|2163|
